O Bitbucket é uma plataforma baseada em nuvem, desenvolvida pela Atlassian, que serve como um repositório para hospedar, gerenciar e colaborar em códigos-fonte usando o sistema de controle de versão Git. 
Em termos básicos, ele funciona como um "guarda-roupa" ou "nuvem" organizado para seus projetos de programação, permitindo que múltiplos desenvolvedores trabalhem no mesmo código sem sobrescrever o trabalho uns dos outro

Issue tracking (rastreamento de problemas) é um processo e sistema de software utilizado para registrar, priorizar, monitorar e resolver bugs, tarefas ou solicitações de suporte. Essencial para o desenvolvimento de software e atendimento ao cliente, ele centraliza o histórico de atividades, permitindo que equipes colaborem na resolução de problemas de forma organizada.

Um Pull Request (PR) é uma solicitação formal feita por um desenvolvedor para integrar (mesclar/merge) as alterações de código feitas em um branch (ramo) de desenvolvimento para o branch principal (como main ou master) de um projeto. Ele serve como um espaço para revisão, discussão e aprovação de código antes da mesclagem, sendo essencial para colaboração em equipe
